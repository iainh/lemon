A command-line frontend for macOS Virtualization.framework, written in Zig.

## Requirements

- macOS 12.0+ (Monterey or later)
- Nix with flakes enabled

## Development

Enter the development environment:

```bash
nix develop
```

Build:

```bash
zig build
```

Run:

```bash
zig build run
```

Test:

```bash
zig build test
```
